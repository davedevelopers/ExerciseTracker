## Getting Started

Include a `.env` file in the `root/backend` directory with the following environment variables.

```
ATLAS_URI = 'mongodb://localhost/<DATABASE_NAME>'
```

## Built with

* nodejs
* express
* mongodb
* mongoose
* react
